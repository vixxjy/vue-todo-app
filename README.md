## Vue-todo-App

This app was built with Vue, sass, and animate css kindly drop suggestions and tips:

### `Installation`

- $ git clone the repo
- $ cd into repo
- $ npm install
- $ npm run dev

Then navigate to http://localhost:8080 to view the demo.
